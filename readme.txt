This plugin convert current web page to pdf file

== Description ==

This plugin convert current web page to pdf file. It is handy to print, send via email and maintain when web page saved into single file with cross platform format. 

== Frequently Asked Questions ==

= How do i use this plugin.

This is simple widget plugin. Make standard installation Plugins -> Add New. Then drag widget to desired place using Appearance -> Widget.

== Installation ==

1. Install plugin via administrative console.
2. Enable Web2PDF Converter widget using Widgets administrative page.
